# Online Examination System 

The Online Examination System is a Java-based console application that allows users to log in, update their profile, change their password, attend a Multiple-Choice Question (MCQ) exam and automatically submit answers when the timer ends. This project demonstrates core Java concepts such as classes, objects, methods, loops, validation, and working with collections.

## Features
### Feature | Description
User Login | Secure username & password based login
Profile Update |	Modify name and email
Password Change |	Update old password with validation
MCQ Exam |	Java-related questions displayed one by one
Timer & Auto Submit |	Exam auto-submits when time expires
Score Report |	Displays attempted, correct answers & score
Session Logout | Ends session and exits application

## Technologies Used

### Technology | Purpose
Java | Core programming
IntelliJ IDEA |	Development Environment
Collections |	Store questions
OOP Concepts |	Classes & Objects
